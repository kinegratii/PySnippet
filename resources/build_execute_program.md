# Python打包


目前在windows平台上将Python程序打包成exe文件主要有三个工具。

- py2exe http://www.py2exe.org/
- cx_freeze http://cx-freeze.sourceforge.net/
- PyInstaller http://pythonhosted.org/PyInstaller/
